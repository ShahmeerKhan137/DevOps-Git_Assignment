git clone: this command is used to copy remote repository to your local machine.
git status: this command tells us the status of repo, including modified, staged files etc.
git add: thiss command help us to add the file into staging and make ready for commit.
git commit: this command save the staged file into repo with a proper discriptive comment.
gir branch: it is used to create, del branch.
git checkout: it is used to switch between main branch and the ither branch.
gir merge: it is used to merge the changes from one branch to another.
git pull: it is used to fetch the data from remote repo to your local repo.
git push: this is used to push commit files to remote repo.
